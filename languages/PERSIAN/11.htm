Counting in Persian

 

Persian is written from right to left except when writing numerals and numeric mathematical statements. These are written from left to right as in English.

 

Just as English, to be able to count in Persian, you have to learn the numbers from 1 to 20 first. Counting thereafter becomes like a repeating process where you only need to know about the following numbers in order to be able to call any number between:

30, 40, 50, 60, 70, 80, 90,

100, 200, 300, 400, 500, 600, 700, 800, 900

1000

1000000

...

To call any number after 20, you must pronounce the sound "o" after each digit (beside the last one).

To understand this better, lets apply this rule to reading English numbers.

Assuming that the sound "o" must me pronounced after each digit,

the number 243 will be read: two hundred o forty o three.

the number 4679 will be read: four thousand o six hundred o seventy o nine

 

the "o" sound in Persian is "و "

 

Now lets read 21 in Persian

twenty in Persian is: بیست  [bist],   

and one is: یک  [yek]

so twenty one is written:   بیست و یک    [bist o yek]

 

 

Now lets read 7592 in Persian

seven thousand in Persian is: هفت هزار   [haft hezar]

five hundred in Persian is: پانصد   [pansad]

ninety in Persian is:  نود  [navad]

two in Persian is:  دو  [do]

 
so seven thousand five hundred and ninety two is written:

 

هفت هزار و پانصد و نود و دو

   [haft hezar o pansad o navad o do]
