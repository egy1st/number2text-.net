Overview

DC Number2Text
 DC Number2Text is one of the first programs to offer numeric translation for program development. It converts any number string in your code into alphabetic text, with accurate place values from the ones digit onwards.

Great for banks billing
 This software is useful when making financial reports, generating bills, and printing checks. There’s no need to translate and type in each number – just run Number2Text and a text translation will appear next to your figures. Since encoding is automatic, there is no room for fraud or human error.

Multiple language support
 Number2Text can translate numbers in English, Arabic, French, and German. It can also translate converted text between these four languages – a valuable feature for online or international business.  EgyFirst plans to add more languages according to user demand. You can offer your own suggestions at support@egyfirst.com.

Currency options
 Number2Text sets the currency depending on the language you choose. The default currency is dollar for the English translator, euro for French and German, and piaster for Arabic. The pound is also supported. Standard fields allow you to set singular and plural units (dollar/dollars) and assign currency symbols.

Simple syntax
Number2Text uses very simple codes to set and apply your preferences. All the commands fit into a few lines of code, so it won’t clutter your program source like other add-ons. It also comes with a free tutorial to familiarize you with the basic syntax.

Works with any application
 Number2Text is a .Net class library tht works with any development applications such as Borland C++. Delphi, Visual C++, and Visual Basic. It automatically generates code to display special characters for foreign language translations.

Easy setup and installation
 Number2Text installs in less than a minute on Windows 98, NT, 2000, and XP. During installation, the file “DC_Num2Text10.dll” is registered on C:\Windows\system32, although you can specify another path. You can also set it to run automatically when you open your application – just add the DLL file as a reference from the Project menu
