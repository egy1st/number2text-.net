setRegistrationKey Function

setRegistrationKey Function enables you to set your purchased serial number, to get a complete use of your software. if you deployed an application that use Dc Number2Text without setting your registration key, this will result in a limited version that limit number to only 10000 while with registered version you can use number up to one Billion (999999999.99).

Syntax:

Public Sub setRegistrationKey(ByVal myKey As String)

Example:

oTextNum.setRegistrationKey("0000-000-000-000-0000")

 

Note:

This function should be used immediately after your initiate your instance of DC Number2Text

Example:

Dim oTextNum As New DynamicComponents.Number2Text()

oTextNum.setRegistrationKey("0000-000-000-000-0000") ' replace with your purchased serial number once you get one

oTextNum.translateNumber(123656.23)
