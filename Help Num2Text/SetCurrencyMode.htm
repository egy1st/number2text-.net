setCurrencyMode Function
setCurrencyMode function enables you to customize your currency text among three types: Type, Sign and Abreviation.

Syntax:

Public Sub setCurrencyMode(ByVal mode As Currency_Mode)

Currency_Mode takes one the following values:
Text = 1                   //  i.e. dollar
Sign = 2                   // i.e. $ 
Abreviation = 3        // i.e. USD

Example:

oTextNum.setCurrencyMode(DynamicComponents.Num2Text.Currency_Mode.Text)

when you use TtranslateNumber function , result may be something like that
four hundreds and twenty one dollars and twenty three cent

Currency can be used in 3 modes by the use of setCurrencyMode
