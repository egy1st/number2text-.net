Features

DC Number2Text is one of the first programs to offer numeric translation for program development. It converts any number string in your code into alphabetic text, with accurate place values from the ones digit onwards. for now 13 languages are supported so far and counting.

Example:

Dim oTextNum As New DynamicComponents.Number2Text()

dim strNum as string

strNum  = oTextNum.TranslateNumber("159584.58", Number2Text.Language_ID.English) // translate into English
